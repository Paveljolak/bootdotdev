# Bookbot

First python project [Bookbot](https://github.com/Paveljolak/bootdotdev/tree/main/bookbot).

Analyzes the word count and the frequency of letters inside a certain booktext. 

Works as a CLI command.

## Prerequisites

- Installed version of python3

## How to run

```bash

# Clone the github repo
git clone <github-project-url>

# Go into the bookbot directory
cd bookbot

# Run the main.py 
python3 main.py books/<choose_a_book> 

# EXAMPLE:
python3 main.py books/frankenstein.txt

```
